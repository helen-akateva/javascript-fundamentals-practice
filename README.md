The project includes three practical tasks:

### Task 1: Transaction Calculator
- Calculates the total price of droids based on quantity and price per unit
- Uses template literals for formatted output
- Example: `makeTransaction(5, 3000)` returns `"You ordered 5 droids worth 15000 credits!"`

### Task 2: Shipping Cost Calculator
- Computes total shipping cost including delivery fees
- Handles international shipping calculations
- Example: `getShippingMessage('Australia', 120, 50)` returns `"Shipping to Australia will cost 170 credits"`

### Task 3: Element Width Calculator
- Calculates the total width of an element including content, padding, and border
- Demonstrates string to number conversion with `parseFloat()`
- Example: `getElementWidth('50px', '8px', '4px')` returns `74`
